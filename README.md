# Chronosync Sweet Chicago Theme

## Linux Distro
Arch Linux with KDE Plasma

## Prerequisites
KDE Plasma
Fastfetch

## KDE Themes used
Revolutionary 95
Sweet

## Steps to install

##### Step 1:
Revolutionary 95: Apply every setting except window layout

##### Step 2:
Sweet: Apply every setting except icons, window decorations and cursors

##### Step 3:
Copy fastfetch config files and onfigure panels as you
like!
